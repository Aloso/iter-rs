# Iterator

View this website on <https://iter-rs.herokuapp.com>.

## Run locally

```sh
npm install
npm start
```

## Build

```sh
npm run clean
npm run build
```

## Publish

```sh
# login on heroku cli, then build the app
# then
heroku plugins:install heroku-cli-static
heroku static:deploy --app=iter-rs
```
